You

dbatools has been tested in a lot of environments, but you may do things differently than we do. Maybe you use indexes and we don’t (kidding ;)). We want dbatools to work for all of us, and would love to hear from you if you find a scenario that is not covered.

As of today, dbatools has over 140 contributors on github and over 1700 users in the Slack channel, many of whom help with module design, coding, Q&A and more. If you’re a PowerShell developer or even if you want to learn, we can guide you; please consider contributing to the project.
